Tip: to use Private Label on the live site, set use_private_label: true in assets/data/content.json.

Quick test checklist

Do this in order:

  1. Create a subdomain like book.YOURDOMAIN.com in your DNS (CNAME).
  2. In Beds24, enable Private Label and paste that subdomain.
  3. After Beds24 shows it as active, set beds24_private_label_url to https://book.YOURDOMAIN.com/book-calypsophuket.
  4. Set beds24_css_url to your live CSS file (e.g. https://YOURDOMAIN.com/beds24.css).
  5. Turn on use_private_label: true and re-deploy.

If the booking page URL in the browser stays on your subdomain the entire time, you’re good.
