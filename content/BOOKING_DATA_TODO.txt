BOOKING DATA (MANUAL COPY)

Booking.com blocks automated scraping, so this template expects you to paste the details manually.

Edit: content/apartments/apartma1.json

Recommended fields to update:
- name
- teaser
- landingTitle
- landingSubtitle
- ratingLine
- size, maxGuests, beds
- facts: sleeps / bedrooms / bathrooms / wifi
- whyLove: bullet points
- amenities: add inside existing fields or extend
- neighborhood.mapEmbedUrl (Google Maps → Share → Embed a map → copy iframe src)
- testimonial.quote / author / stars

Images:
- Hero video: content/pages/home.json → hero.video
- Hero poster: assets/media/images/hero.jpg
- Gallery: assets/media/images/01.jpg … 44.jpg
